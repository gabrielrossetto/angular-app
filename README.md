# AngularApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.7.

## Development server

Run `npm start` for a dev server. Navigate to `http://localhost:4200/dashboard/users`. The application will automatically reload if you change any of the source files.
